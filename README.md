# Property Management — Landing Page

A modern property management landing page built with **Next.js**, converted from a Figma design as part of the OneLittleWeb Web Developer hiring task.

🔗 **Live Site:** [https://property-manager-ruddy-delta.vercel.app/](https://property-manager-ruddy-delta.vercel.app/)

---

## Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- SSG (Static Site Generation) & CSR (Client-Side Rendering)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

### 3. Run the development server

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.
